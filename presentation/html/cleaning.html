Cleaning the raw data

First, I identidied stock tickers original posts and comments refer to.

Then I downloaded stock quotes from Yahoo.com

Next I kept only stocks that are priced at $5 or less.

I aggregeted comments and posts, marking original posts with a dummy variable.

For each stock I calculated weekly returns, and a boolean variable, indicating weeks with abnormal returns. These boolean variables return true if the weekly return (or maximum weekly return) deviate from the rolling average weekly return by more than 2 standard deviations.

\begin{cases} Flag_{up} = 1 \space\text{if} \space R > 2\overline{\sigma}\space *\space max(0,\overline{R}) \\ Flag_{up} = 0 \space\text{otherwise} \end{cases}


\(\overline{R} = \frac{max(P) - P_0}{P_0}\)

For each comment I calculate polarity score, using TextBlob package.
